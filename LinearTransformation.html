Section 1.8: Introduction to Linear Transformations

Concept 1: Matrix Transformations T(x) = Ax

Definition: Matrix Transformation

A matrix transformation is a function T from ℝⁿ to ℝᵐ defined by T(x) = Ax, where A is an m × n matrix, x is in ℝⁿ, and Ax is the matrix-vector product.

T: ℝⁿ → ℝᵐ defined by T(x) = Ax

Key Points:

  • The domain of T is ℝⁿ (when A has n columns)
  • The codomain of T is ℝᵐ (when A has m rows)
  • For each x in ℝⁿ, T(x) is computed as Ax
  • The range of T is the set of all linear combinations of columns of A

Example:

Let A = \(\begin{pmatrix}1 & -3 \\ 3 & 5 \\ -1 & 7\end{pmatrix}\), u = \(\begin{pmatrix}2 \\ -1\end{pmatrix}\), b = \(\begin{pmatrix}3 \\ 2 \\ -5\end{pmatrix}\)

Define T: ℝ² → ℝ³ by T(x) = Ax

Then T(x) = \(\begin{pmatrix}1 & -3 \\ 3 & 5 \\ -1 & 7\end{pmatrix}\)\(\begin{pmatrix}x₁ \\ x₂\end{pmatrix}\) = \(\begin{pmatrix}x₁ & - & 3x₂ \\ 3x₁ & + & 5x₂ \\ -x₁ & + & 7x₂\end{pmatrix}\)

Concept 2: Linear Transformation Definition

Definition: Linear Transformation

A transformation (or mapping) T is linear if:

(i) T(u + v) = T(u) + T(v) for all u, v in the domain of T
(ii) T(cu) = cT(u) for all scalars c and all u in the domain of T

Properties:

  • Property (i): Preserves vector addition
  • Property (ii): Preserves scalar multiplication
  • Every matrix transformation is a linear transformation
  • If T is linear, then T(0) = 0
  • If T is linear, then T(cu + dv) = cT(u) + dT(v)

Important Consequences

If T is a linear transformation, then:

T(0) = 0
T(cu + dv) = cT(u) + dT(v)
T(c₁v₁ + c₂v₂ + ... + cₚvₚ) = c₁T(v₁) + c₂T(v₂) + ... + cₚT(vₚ)

Concept 3: Domain, Codomain, Range, and Image

Definitions

For a transformation T: ℝⁿ → ℝᵐ:

  • Domain: The set of all possible inputs. For T: ℝⁿ → ℝᵐ, the domain is ℝⁿ
  • Codomain: The set where outputs are expected to lie. For T: ℝⁿ → ℝᵐ, the codomain is ℝᵐ
  • Image of x: For x in ℝⁿ, the vector T(x) in ℝᵐ is called the image of x
  • Range: The set of all images T(x), denoted Range(T) = {T(x) : x ∈ ℝⁿ}
Range(T) = {b ∈ ℝᵐ : b = T(x) for some x ∈ ℝⁿ}

Key Relationship: For matrix transformation T(x) = Ax, Range(T) = Col(A)

Important Distinction:

The range is a subset of the codomain. The range consists of all vectors that are actually outputs of T, while the codomain is the entire space where outputs could potentially lie.

Concept 4: Properties of Linear Transformations

Key Properties

If T: ℝⁿ → ℝᵐ is a linear transformation, then:

  1. Zero vector property: T(0) = 0
  2. Linear combination property: T(c₁v₁ + c₂v₂ + ... + cₚvₚ) = c₁T(v₁) + c₂T(v₂) + ... + cₚT(vₚ)
  3. Preservation of linear dependence: If {v₁, v₂, ..., vₚ} is linearly dependent, then {T(v₁), T(v₂), ..., T(vₚ)} is linearly dependent
  4. Subspace preservation: If V is a subspace of ℝⁿ, then T(V) = {T(v) : v ∈ V} is a subspace of ℝᵐ
Important Note:

The converse of property 3 is NOT true: linearly independent vectors may map to linearly dependent vectors under a linear transformation.

Section 1.9: The Matrix of a Linear Transformation

Concept 1: Standard Matrix of a Linear Transformation

Theorem 10: Existence of Standard Matrix

Let T: ℝⁿ → ℝᵐ be a linear transformation. Then there exists a unique matrix A such that:

T(x) = Ax for all x in ℝⁿ

In fact, A is the m × n matrix whose jth column is the vector T(e_j), where e_j is the jth column of the identity matrix in ℝⁿ:

A = \(\begin{pmatrix}T(e₁) & T(e₂) & ... & T(eₙ)\end{pmatrix}\)

This matrix A is called the standard matrix for the linear transformation T.

Key Insight:

Every linear transformation from ℝⁿ to ℝᵐ can be viewed as a matrix transformation, and vice versa. The term "linear transformation" focuses on a property of a mapping, while "matrix transformation" describes how such a mapping is implemented.

Concept 5: Rotations

Rotation Transformations

Standard Matrix for Rotation:

For rotation by angle φ (counterclockwise) about the origin in ℝ²:

R(φ) = \(\begin{pmatrix}cos(φ) & -sin(φ) \\ sin(φ) & cos(φ)\end{pmatrix}\)

Common Rotation Angles:

  • 90° (π/2): \(\begin{pmatrix}0 & -1 \\ 1 & 0\end{pmatrix}\)
  • 180° (π): \(\begin{pmatrix}-1 & 0 \\ 0 & -1\end{pmatrix}\)
  • 270° (3π/2): \(\begin{pmatrix}0 & 1 \\ -1 & 0\end{pmatrix}\)

Properties of Rotations:

  • Rotations preserve distances: ||R(x)|| = ||x||
  • Rotations preserve angles between vectors
  • Composition of rotations: R(φ)R(θ) = R(φ + θ)
  • Inverse rotation: R(φ)⁻¹ = R(-φ) = R(φ)ᵀ
  • det(R(φ)) = 1 for all φ
Clockwise vs Counterclockwise:

The standard formula gives counterclockwise rotation. For clockwise rotation by angle φ, use R(-φ) or equivalently \(\begin{pmatrix}cos(φ) & sin(φ) \\ -sin(φ) & cos(φ)\end{pmatrix}\).
